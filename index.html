    Como usar o modelo

    Cada linha da planilha representa 1 contrato. A primeira linha é o cabeçalho — não edite. Preencha a partir da linha 2. Salve no formato .xlsx ou .csv.

    Arquivo Word com os placeholders que será preenchido automaticamente

    modelo-contrato.docx Carregue o arquivo .docx com os placeholders {{campo}}
    Como preparar o modelo .docx

    No seu Word, substitua os campos variáveis pelos placeholders entre chaves duplas. Ex: onde estava o nome do cliente, coloque {{razao_social}}. Veja a lista completa de placeholders abaixo.
